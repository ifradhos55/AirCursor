# Air-Cursor

This is An Air Gesture based Cursor using MediaPipe and OpenCV.

## How to run

To run the program, use the following command:

python testhands.py

It takes a few second to initially launch so just wait for it.

## How to use

First you need to install these libraries and frameworks:
opencv, mediapipe, pyautogui, numpy, PyQt5 and pynput.

Use this command to install them:

pip install opencv-python mediapipe pyautogui numpy PyQt5 pynput

After they are installed you can move around with your index and pinch your index with your thumb to click on something.

Thumbs up will scroll up and thumbs down will scroll down.
